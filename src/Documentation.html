Документация по SmartCalc v1.0

Часть 1. Реализация SmartCalc

  1. Язык программирования и среда разработки

    • Программа разработана на языке C стандарта C11 с использованием компилятора gcc.
    • Используется библиотека и модули QT.

  2. Структура проекта

    • Код программы находится в папке src.
    • Сборка программы настроена с помощью Makefile, включая стандартные цели для GNU-программ.

  3. Принципы разработки

    • Разработка ведется согласно принципам структурного программирования.
    • Код соответствует Google Style.

  4. Тестирование

    • Обеспечено покрытие unit-тестами модулей, связанных с вычислением выражений, с использованием библиотеки Check.

  5. Графический интерфейс

    • Реализован графический пользовательский интерфейс.

  6. Функциональность калькулятора

    • Программа поддерживает ввод как целых, так и вещественных чисел.
    • Вычисление производится после полного ввода выражения и нажатия на символ "=".
    • Поддерживает вычисление произвольных скобочных арифметических выражений в инфиксной нотации.
    • Возможность подстановки значения переменной x в виде числа.
    • Построение графика функции по заданному выражению с переменной x.
    • Пользователь может вводить до 255 символов.

Часть 2. Кредитный калькулятор

  1. Функциональность кредитного калькулятора

    • Реализован специальный режим "кредитный калькулятор".
    • Входные параметры: общая сумма кредита, срок, процентная ставка, тип кредита (аннуитетный, дифференцированный).
    • Вывод: ежемесячный платеж, переплата по кредиту, общая выплата.
